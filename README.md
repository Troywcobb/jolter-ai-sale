## Contact email requirement

The page uses:

`domain-offers@jolter.ai`

Before pointing public traffic here, configure NameCheap email forwarding for that address or replace it in `index.html` with the preferred inbox.

## Deploy options

### Option A — NameCheap hosting / cPanel
Upload `index.html` to the document root for `jolter.ai`.

### Option B — Cloudflare Pages
1. Create a Cloudflare Pages project.
2. Upload this folder as a static site.
3. Add custom domain `jolter.ai`.
4. Update NameCheap nameservers or DNS as Cloudflare instructs.

### Option C — Netlify/Vercel
Deploy this folder as a static site and add custom domain `jolter.ai`.

## DNS minimum

Point apex `jolter.ai` and `www.jolter.ai` to the chosen hosting provider.
